# ✈️ Flight Booking System

This is a full-featured **Flight Booking System** web application designed for booking, managing, and viewing flight details. It includes both a professional frontend interface and a functional backend with database connectivity.

## 🌐 Technologies Used

- **Frontend**:
  - HTML
  - CSS
  - JavaScript
  - TypeScript
  - React (with `.tsx` files)
- **Backend**:
  - PHP
- **Database**:
  - MySQL (via `.sql` script)
- **Others**:
  - JSON (data/config)
  - SVG/ICO (icons/images)

## 📁 Project Structure

- `index.html` – Homepage entry
- `*.tsx` – React components (frontend UI)
- `*.php` – Server-side scripts
- `*.sql` – Database schema or sample data
- `*.css` – Styling for UI
- `*.json` – Configuration or local data
- `*.js`/`*.ts` – Additional scripting logic

## 🚀 How to Run

1. **Frontend**:
   - Open `index.html` or run via a local development server like Vite or React's dev server.

2. **Backend**:
   - Place the project in `htdocs` of **XAMPP** or a PHP server.
   - Start Apache and MySQL.
   - Import the `.sql` file into **phpMyAdmin** to set up the database.

## 🗃️ Features

- View available flights
- Book tickets with seat preferences
- Manage passenger and payment information
- Display ticket pricing
- Admin-side: Issue/cancel tickets, manage routes, view manifests

## 🧑‍💻 Developer

This project is part of a professional flight booking system with complete frontend and backend support.
